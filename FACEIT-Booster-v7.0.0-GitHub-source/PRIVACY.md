# FACEIT Booster Privacy Policy

Last updated: July 14, 2026

FACEIT Booster stores extension preferences, including an optional FACEIT nickname, in Chrome's synchronized extension storage. These settings are used only to provide the features selected by the user.

When a user opens or analyzes a FACEIT matchroom, the extension sends the relevant FACEIT match ID, player ID, or nickname to the FACEIT Booster backend. The backend uses those identifiers to request the corresponding public game information from the official FACEIT API. The backend does not receive or store the user's FACEIT password, browser history, payment information, or FACEIT API key.

The backend may temporarily cache FACEIT responses to improve speed and reduce API usage. Hosting providers may retain standard technical logs such as IP address, request time, and requested path according to their own retention policies.

FACEIT Booster does not sell personal information, use it for advertising, or transfer it for unrelated purposes.

Users can remove locally stored extension settings by uninstalling the extension or clearing its data in Chrome. Privacy questions can be directed to the project owner through the support contact listed on the Chrome Web Store page.
